# The Colonel's Battleground — Australian QSR Deep Research for KFC Australia

An interactive, single-page research report on Australia's quick service restaurant (QSR / fast food) industry, prepared as strategy support for KFC Australia leadership (July 2026).

**Live report:** open `index.html` (or visit the GitHub Pages URL once Pages is enabled).

## Contents
1. Executive summary
2. Market landscape (A$29.7bn industry, chicken category, store networks, new entrants)
3. KFC Australia today (Collins Foods FY24–FY26 financials, digital mix, brand health)
4. The competitive set (McDonald's, Hungry Jack's, Guzman y Gomez, Red Rooster, Domino's, Subway, Wendy's/Popeyes)
5. Pricing & the value war (2026 value ladder across chains)
6. Consumer & channel trends (delivery duopoly post-Menulog, daypart growth, digital behaviour)
7. Cost & regulatory environment (FWC wage reviews 2023–2026, compliance)
8. SWOT assessment
9. Strategic recommendations (six moves for FY27–FY29)
10. Risks, sources & disclaimer

## Tech
- Single static `index.html` — no build step, works on GitHub Pages as-is
- Interactive charts via [Apache ECharts](https://echarts.apache.org/) (CDN)
- Fonts: Fraunces / Inter / IBM Plex Mono (Google Fonts)

## Deploying to GitHub Pages
Repository is pre-configured: Pages can serve directly from the `main` branch root.

1. Go to **Settings → Pages**
2. Under **Build and deployment**, choose **Source: Deploy from a branch**
3. Select branch **main**, folder **/ (root)**, save
4. The site will be live at `https://<username>.github.io/<repo-name>/` within a minute or two
